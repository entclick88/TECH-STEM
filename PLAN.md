# แผนงานฝ่ายเทคโนโลยีสารสนเทศและสเต็มศึกษา (Tech & STEM Division)
โรงเรียน มอ.วิทยานุสรณ์ — ปีการศึกษา 2569
ฉบับควบรวมงานเทคโนโลยีสารสนเทศ + งานสเต็มศึกษา

---

## 1. หลักการและเหตุผล

ควบรวมงานด้านเทคโนโลยีสารสนเทศ (Network, School Connex, พัฒนาระบบ) เข้ากับงานสเต็มศึกษา
(Fabrication Lab, Smart Farm) ให้เป็นฝ่ายเดียว เพื่อ:

- ลดความซ้ำซ้อนของบุคลากรและงบประมาณ ใช้ทักษะร่วมกันได้ (โค้ด + ฮาร์ดแวร์ + ผลิตชิ้นงาน)
- สร้างโมเดล **"หน่วยงานเลี้ยงตัวเองได้"** — ใช้เครื่องมือ STEM ผลิตชิ้นงาน/บริการวิชาการสร้างรายได้
  เป็นต้นแบบให้หน่วยงานอื่นของโรงเรียน
- ให้นักเรียนมีส่วนร่วม **ทุกขั้นตอน**: ออกแบบ → ผลิต → ตลาด → ขาย → บัญชี (เรียนรู้แบบ Real Business)

## 2. วิสัยทัศน์

> "ศูนย์เทคโนโลยีและนวัตกรรมที่ขับเคลื่อนโรงเรียนด้วยระบบดิจิทัล และเลี้ยงตัวเองได้ด้วยผลงานนักเรียน"

## 3. โครงสร้างงาน 4 กลุ่มงาน

### กลุ่มงานที่ 1 — โครงสร้างพื้นฐานและเครือข่าย (Network & Infrastructure)
- ดูแลอินเทอร์เน็ต, Wi-Fi, Network ทั้งโรงเรียน
- ทะเบียนครุภัณฑ์ไอที + ระบบแจ้งซ่อมออนไลน์ (พัฒนาต่อจากแพทเทิร์น EQUIPMENT-LOAN)
- แผนบำรุงรักษาเชิงป้องกัน (PM) รายภาคเรียน + มาตรฐานความปลอดภัยไซเบอร์

### กลุ่มงานที่ 2 — School Connex และระบบบริการ
- ดูแลการใช้งาน School Connex ของนักเรียนและบุคลากร (บัญชีผู้ใช้, อบรม, คู่มือ, Helpdesk)
- จุดรับเรื่อง (Ticket) ช่องทางเดียว ผ่านเว็บฝ่าย

### กลุ่มงานที่ 3 — พัฒนาระบบดิจิทัล (Software Development)
- ดูแล/ต่อยอดระบบที่มีอยู่: ระบบสารสนเทศ รร (แชทบอท+ฟอร์ม), บันทึกข้อความอัตโนมัติ,
  ยืม-คืนอุปกรณ์ (EQUIPMENT-LOAN), ประกาศผลสอบ (MIDTERM-RESULTS), รับสมัครนักเรียน (student-admission),
  แบบสำรวจ (inspire-space-survey), Portfolio นักเรียน
- มาตรฐานเทคโนโลยีกลาง: หน้าเว็บ static (GitHub Pages/Cloudflare Pages) + Cloudflare Worker + D1
- ระบบใหม่ปีนี้: **เว็บฝ่าย Tech & STEM** (โปรเจกต์นี้) + **ร้านค้าออนไลน์ STEM Shop** + **ระบบจองใช้เครื่อง/บริการวิชาการ**

### กลุ่มงานที่ 4 — STEM Innovation Lab (สร้างรายได้)
เครื่องมือ: 3D Printer, Laser Cutter, CNC, อุปกรณ์อิเล็กทรอนิกส์/IoT ฯลฯ

สายรายได้ 4 ทาง:
1. **ผลิตชิ้นงานตามสั่ง (Custom Fabrication)** — ป้ายชื่อ, ถ้วยรางวัล, ของที่ระลึกโรงเรียน, งานตัด/แกะสลัก
2. **สินค้าแบรนด์โรงเรียน (School Merch)** — ออกแบบโดยนักเรียน ผลิตเป็นล็อต ขายผ่านหน้าร้านออนไลน์
3. **บริการวิชาการ** — คอร์สอบรม 3D/Laser/CNC/Coding/IoT ให้โรงเรียนอื่น หน่วยงาน และบุคคลทั่วไป (คิดค่าลงทะเบียน)
4. **Smart Farm** — ผลผลิตจากฟาร์มอัจฉริยะ (ต่อยอดระบบ BARNMAN) + ชุดคิท Smart Farm ขนาดเล็กสำหรับขาย/อบรม

บทบาทนักเรียน (ทุกขั้นตอน):
| ขั้นตอน | บทบาทนักเรียน | ทักษะที่ได้ |
|---|---|---|
| ออกแบบ | ออกแบบผลิตภัณฑ์ด้วย CAD/กราฟิก | Design Thinking, CAD |
| ผลิต | คุมเครื่อง 3D/Laser/CNC (ภายใต้ครูกำกับ) | วิศวกรรม, ความปลอดภัย |
| การตลาด | ถ่ายภาพ, ทำคอนเทนต์, ดูแลเพจ/ร้านออนไลน์ | Digital Marketing |
| ขาย/บริการ | รับออเดอร์, แพ็ก-ส่ง, เป็นผู้ช่วยวิทยากรอบรม | บริการลูกค้า, สื่อสาร |
| บัญชี | บันทึกรายรับ-จ่าย, สรุปงบรายเดือน | การเงิน, ผู้ประกอบการ |

การเงิน: รายรับเข้าระบบเงินรายได้สถานศึกษาตามระเบียบ แบ่งสัดส่วน เช่น
ต้นทุนวัสดุ/ซ่อมบำรุงเครื่อง 40% • กองทุนพัฒนาห้อง STEM 30% • ค่าตอบแทน/ทุนนักเรียนผู้ปฏิบัติงาน 20% • สำรอง 10%
(สัดส่วนจริงให้เป็นไปตามระเบียบการเงินของโรงเรียน)

## 4. ระบบเว็บฝ่าย (โปรเจกต์นี้ — TECH-STEM)

- **Phase 1 (ทำแล้วในโฟลเดอร์นี้):** เว็บฝ่ายหน้าเดียว สไตล์โมเดิร์น AI-Tech (docs/index.html)
  รวมลิงก์ทุกระบบ, แนะนำ STEM Lab, โชว์สินค้า/บริการ, ช่องทางติดต่อ — deploy ด้วย GitHub Pages
- **Phase 2:** STEM Shop จริง — Cloudflare Worker + D1 (ตามแพทเทิร์น MIDTERM-RESULTS/EQUIPMENT-LOAN)
  ตาราง products, orders, bookings; หน้าแอดมินให้นักเรียนฝ่ายขายจัดการสต๊อก/ออเดอร์
- **Phase 3:** ระบบจองใช้เครื่อง + จองคอร์สอบรม + Dashboard รายได้แบบ real-time โชว์หน้าเว็บ
- **Phase 4:** เชื่อม Smart Farm (BARNMAN) แสดงข้อมูลฟาร์ม + สั่งจองผลผลิตล่วงหน้า

## 6. ตัวชี้วัด (KPI) ปีแรก

- ระบบเครือข่าย uptime ≥ 99% / ปิด ticket แจ้งซ่อมเฉลี่ย ≤ 3 วันทำการ
- ผู้ใช้ School Connex active ≥ 90% ของนักเรียน+บุคลากร
- รายได้รวม STEM Lab ≥ ครอบคลุมค่าวัสดุสิ้นเปลือง + ค่าบำรุงเครื่องทั้งปี (เลี้ยงตัวเองขั้นต่ำ)
- นักเรียนผ่านการปฏิบัติงานจริงใน STEM Crew ≥ 30 คน/ปี
- บริการวิชาการ ≥ 4 คอร์ส/ปี
- ยอดขายออนไลน์ ≥ 100 ออเดอร์ในปีแรก

## 7. ความเสี่ยงและการรับมือ

- **เครื่องเสีย/ค่าซ่อมสูง** → กันเงิน 40% เข้าค่าวัสดุ-ซ่อมบำรุง + ทำตารางบำรุงรักษา
- **นักเรียนติดตารางเรียน** → จัดเวรเป็นทีม, นับเป็นชั่วโมงกิจกรรม/ชุมนุม
- **ความปลอดภัยการใช้เครื่อง** → อบรม + สอบใบอนุญาตใช้เครื่อง (License) ก่อนใช้จริง, ครูอยู่กำกับเสมอ
- **ระเบียบการเงิน** → หารือฝ่ายการเงินโรงเรียนตั้งแต่เริ่ม กำหนดเส้นทางเงินให้ถูกระเบียบก่อนเปิดขายภายนอก
